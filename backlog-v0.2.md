# ÉPICOS E HISTÓRIAS DE USUÁRIO DETALHADOS

## ÉPICO 01 - GESTÃO DE NORMAS
Permitir armazenar e gerenciar normas técnicas

### Características
- cadastro de normas
- versionamento de normas
- correlação de requisitos

### Histórias de usuário
**US01**
```
Como gestor do sistema
quero cadastrar uma nova norma
para que ela esteja disponível no sistema
```

**US02**
```
Como gestor do sistema
quero registrar versões de uma norma
para manter histórico e rastreabilidade
```

**US03**
```
Como engenheiro
quero listar normas ativas
para saber quais são válidas para uso
```

**US04**
```
Como engenheiro
quero registrar requisitos associados a uma norma
para estruturar e consultar os requisitos normativos
```

**US05**
```
Como engenheiro
quero registrar correlações entre requisitos
para compreender dependências e referências normativas
```

## ÉPICO 02 - SISTEMA DE TAGS
Permitir classificar normas usando palavras-chave técnicas

### Características
- cadastro de tags
- associação de tags às normas

### Histórias de usuário
**US06**
```
Como gestor do sistema
quero cadastrar novas tags
para classificar normas de forma padronizada
```

**US07**
```
Como gestor do sistema
quero associar tags às normas
para melhorar a organização e busca
```

## ÉPICO 03 - BUSCA E FILTRAGEM
Permitir encontrar normas rapidamente

### Características
- busca por código
- busca por texto
- filtragem por área
- filtragem por tags
- filtros combinados

### Histórias de usuário
**US08**
```
Como engenheiro
quero filtrar normas por tags
para encontrar rapidamente normas relevantes
```

**US09**
```
Como engenheiro
quero pesquisar normas por código ou nome
para encontrar uma norma específica
```

**US10**
```
Como engenheiro
quero filtrar normas por área técnica
para reduzir o número de resultados
```

**US11**
```
Como engenheiro
quero combinar filtros de área e tags
para encontrar normas com mais precisão
```

## ÉPICO 04 - GESTÃO DE USUÁRIOS
Permitir "tipos" de usuário no sistema

### Características
- cadastro de usuários
- gerenciamento de usuários

### Histórias de usuário
**US12**
```
Como gestor do sistema
quero cadastrar novos usuários no sistema
para que eles possam acessar a plataforma conforme suas permissões
```

**US13**
```
Como gestor do sistema
quero visualizar a lista de usuários cadastrados
para que eu possa gerenciar quem tem acesso ao sistema
```

**US14**
```
Como gestor do sistema
quero alterar dados ou permissões de um usuário
para que eu possa ajustar seu acesso quando necessário
```

**US15**
```
Como gestor do sistema
quero desativar usuários
para que eles não tenham mais acesso ao sistema
```

## Sprint 1 - Cadastro e estruturação de normas + Relação entre requisitos

**Meta da Sprint:** US01 e US03

### User Stories da Sprint 1
```
US01 - Cadastrar normas
US02 - Registrar versões de normas
US03 - Listar normas
US04 - Registrar requisitos de uma norma
US05 - Registrar correlações entre requisitos
```

### Resultado da Sprint 1
Permitir cadastrar normas, estruturar requisitos e registrar correlações normativas
